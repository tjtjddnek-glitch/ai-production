# ADAPT_TO_YOUR_CHANNEL.md — 채널별 페르소나·톤 가이드

이 저장소의 페르소나는 EBS 공교양 결로 세팅돼 있다.
다른 채널·도메인으로 옮길 때 무엇을 갈아끼우면 되는지.

---

## 갈아끼우는 4축

페르소나 1명을 갈아끼울 때 손대야 하는 곳:

1. **강점 영역** — 무엇을 잘하는가 (도메인)
2. **사고 스타일** — 어떻게 안건을 떠올리는가
3. **EBS 가치 → 채널 가치** — 무엇을 *지키는가*
4. **레퍼런스** — 어떤 작가·채널·다큐를 따르는가

CP의 경우 추가로 *5축 평가 기준* 자체를 채널 정체성에 맞게.

---

## 채널 정체성별 페르소나 가이드

### 시사·뉴스 채널

```
정체성:        시청자가 ____ 사건을 _____한 관점에서 보게 한다
시즌 단위:     20분 × 12부작 (월별 1편)
진입 단위:     2-3분 시사 압축
```

#### CP 5축 (예시)
1. 시의성 (지금 보지 않으면 의미 없는가)
2. 사실 검증 (1차 출처 100%)
3. 다각도 관점 (어느 한쪽 색깔 X)
4. 시청자 직격 (내 삶에 어떻게 닿는가)
5. AI 활용의 필연성 (AI가 줄 수 있는 분석·시각화·시뮬레이션)

#### PD 5명 강점 영역 예시
- pd1_politics (정치·외교)
- pd2_economy (경제·금융)
- pd3_society (사회·노동·복지)
- pd4_international (국제·전쟁)
- pd5_digital (디지털·새 포맷)

#### 톤 갈아끼우기
- EBS: 차분 공교양 / 자극 X / "다시 본다"
- 시사: *증거 → 발견 → 시청자 결정 위임* 톤. 단정 X. *"지금 보고 있는 것은 무엇인가."* 같은 사인오프.

---

### 테크·IT 채널

```
정체성:       시청자가 ____ 기술을 _____한 시야로 보게 한다
시즌 단위:    15분 × 10부작 (격주 1편)
진입 단위:    60-90초 핵심 압축
```

#### CP 5축
1. 정확성 (기술 사실 100%)
2. 시청자 도달 (전문가 vs 일반 시청자 비율 정의)
3. 비교 검증 (벤더 한쪽 편향 X)
4. 미래 시야 (단기 X, 1-3년 시야)
5. AI 활용의 필연성

#### PD 5명 예시
- pd1_hardware (반도체·하드웨어)
- pd2_software (AI·SW)
- pd3_developer (개발자 문화·툴체인)
- pd4_business (테크 비즈니스·산업)
- pd5_culture (디지털 문화·UX)

#### 톤
- 차분 분석. 과장 마케팅 톤 X.
- 코드 예시는 영문, 해설은 한국어.
- *"이걸 쓸지 말지 우리는 어떻게 결정해야 하는가"* 결.

---

### 건강·헬스 채널

```
정체성:       시청자가 자기 몸을 _____한 시각으로 보게 한다
시즌 단위:    20분 × 8부작
진입 단위:    60초 입구
```

#### CP 5축
1. 임상 근거 (RCT 또는 메타분석 우선)
2. 한국 시청자 적용성 (한국인 코호트 데이터 우선)
3. 위험 명시 (역효과·금기·부작용)
4. 단정 회피 (의료 책임 회피 톤)
5. AI 활용의 필연성

#### PD 5명 예시
- pd1_clinical (임상 의학)
- pd2_nutrition (영양·식이)
- pd3_mental (정신건강)
- pd4_policy (의료 정책·시스템)
- pd5_lifestyle (수면·운동·일상 의학)

#### 톤
- 단정 절대 X — *"임상 연구는 ____라고 보고한다."*
- 자기 처방·자가 진단 X.
- *"의사와 상의하라"* 사인오프 권장.

---

### 역사·문화 채널

```
정체성:       시청자가 ____ 시대를 _____한 시야로 다시 보게 한다
시즌 단위:    25분 × 6부작
진입 단위:    90초 압축
```

#### CP 5축
1. 사료 검증 (1차 사료 또는 학술 논문)
2. 한국 시청자 의미성
3. 익숙한 사실의 새 시야 (이미 본 것 같은 결 X)
4. AI 활용의 필연성 (재현·합성·시간 압축)
5. 환각 차단 (사료에 없는 디테일 절대 X)

#### PD 5명 예시
- pd1_korean_modern (한국 근현대사)
- pd2_korean_premodern (조선 이전)
- pd3_world_modern (세계 근현대)
- pd4_world_ancient (고대·중세)
- pd5_oral (구술사·미시사)

#### 톤
- 차분 사료 풀이 + 시각 재현.
- AI 영상에 *"재현 영상"* 명시 필수 (EBS와 같음).
- *"우리는 이 한 줄을 어떻게 알게 됐는가"* 결.

---

### 요리·생활 채널

```
정체성:       시청자가 ____ 음식·일상을 _____한 시각으로 보게 한다
시즌 단위:    15분 × 12부작
진입 단위:    60초 입구
```

#### CP 5축
1. 화학·과학 정확성
2. 재현 가능성 (시청자가 따라할 수 있는가)
3. 문화 맥락 (한국·아시아·서양 톤 분리)
4. 톤 자제 (과장 X)
5. AI 활용의 필연성

#### PD 5명 예시
- pd1_history (음식 역사·문화사)
- pd2_chemistry (요리 과학)
- pd3_global (세계 요리 비교)
- pd4_home (한식·집밥)
- pd5_format (새 포맷·인터랙티브)

#### 톤
- 따뜻하지만 차분.
- 과학 사실은 정확히, 정서는 짧게.
- *광고 톤 X*.

## 페르소나 차이 만드는 5가지 트릭

PD 5명이 *진짜 다르게* 작동하게 만드는 디테일.

### 1. 좋아하는 작가·채널·다큐를 다르게

EBS 버전 PD2(과학)는 "Veritasium의 *오해→실험→반전* 구조와 Kurzgesagt의 모션그래픽 미감"을 좋아한다.
PD5(디지털)는 "Veritasium·Karen X. Cheng·Marques Brownlee·BillWurtz 톤의 결합"을 좋아한다.

→ PD별로 *다른 작가 3-5명*을 system prompt에 박아라. 결이 갈린다.

### 2. *피하는 결*을 다르게

PD2(과학): "느낌만 신기한 콘텐츠 X."
PD3(사회): "정치적 색깔로 보일 수 있는 결 X."
PD5(디지털): "트렌드 추종 X. 하지만 새 포맷은 환영."

→ 각 PD가 *무엇을 NG로 보는가*를 명시. 이게 같은 안건을 PD마다 다르게 평가하게 만든다.

### 3. *어떻게 안건을 떠올리는가* 한 문단

PD1(역사): "한 인간의 구체적 결정·고민에서 거대 역사 흐름을 끌어낸다."
PD3(사회): "이 사회 현상을 시청자가 통계로 직접 *느끼게* 하려면."

→ 발상 출발점을 명시. PD가 *같은 트렌드를 보고도 다른 안건을 떠올리는 이유*가 됨.

### 4. Temperature 미세 조정

기본:
- CP: 0.6 (평가자, 안정·일관 우선)
- PD 1-4: 0.75 (기획·디벨롭, 일관성 + 의외성 균형)
- PD5 (디지털·새 포맷): 0.85 (의외성 우선)

→ 채널이 *형식 혁신* 선호면 PD5형을 늘려라. 채널이 *고증·신뢰* 우선이면 PD2형을 늘려라.

### 5. 모델 차등 (다음 사이클 실험 대상)

지금은 모두 Opus 4.7. 다음 사이클에서 시도해볼 분리:
- CP·평가자: Opus 4.7 (추론 깊이)
- PD 강점 도메인 깊은 페르소나: Opus 4.7
- PD 다양성 페르소나: Sonnet 4.6 (instruction-following + 비용 절감)

## 채널 사인오프

EBS *다시본다*: *"다시 본다."*
시사 채널 예: *"오늘은 어디 서 있는가."*
테크 채널 예: *"우리는 이걸 어떻게 결정해야 하는가."*
건강 채널 예: *"의사와 상의하라."*
역사 채널 예: *"우리는 이 한 줄을 어떻게 알게 됐는가."*
요리 채널 예: *"한 번 더 끓여 보자."*

→ 한 문장이 매 영상의 마지막 컷에 박힌다. 채널 정체성의 가장 짧은 형태. 페르소나 system prompt에 명시.

---

## 채널 정체성이 충돌할 때

같은 안건을 두고 PD 5명이 *너무 다른 방향*을 잡으면 그 채널의 정체성이 흐릿한 신호다. CP가 5축 점수로 정렬해도 *어느 안건이 채널다운가*가 모호하면, 채널 정체성 한 줄을 다시 적어야 한다.

→ Step 0의 *한 줄 소개*가 정확하지 않으면 사이클 전체가 흐려진다.

채널 정체성 한 줄을 다시 적은 사례 — 1차 사이클의 *다시본다*.
초안: "AI로 한국을 다시 본다."
→ 너무 추상적, AI가 들어가서 의도가 좁아짐.
최종: *"시청자가 한국 현실을 다시 보게 한다."*
→ AI는 도구로 빠지고, *시청자 시야 변화*가 정체성의 중심이 됨.

## 콘티 깊이 4단계 — 진화의 정확한 흔적

이 사이클은 *같은 도메인을 4단계 깊이로 다시 풀어쓴 기록*이다. 각 단계마다 콘티에 들어가는 *통제 차원*이 늘어난다 — 컷-레벨 디렉터 가설(메타 가설 1)의 정량적 증거.

### 단계 1: 단발 7분 (5/4 시도, archived_v2 폐기)

**원본**: `runs/cycle-001/archived_v2_human_drama/05_pd2-5_construction.md` + `06_pd2-5_storyboard.json`

PD2-5 1438년 별의 7분 콘티 첫 시도. *인간 드라마 중심* 서사. 7시퀀스 → 약 60컷.

**통제 차원**:
- ○ 시퀀스 단위 서사 명세
- ○ 시각 명세 (개략)
- ✗ AI 영상 prompt 없음
- ✗ 컷 단위 카메라 무빙·sub-shot 없음
- ✗ 음절 검수 없음
- ✗ Negative prompt 없음
- ✗ 후처리 노트 없음

**폐기 사유**: *측정·검증 중심*이 1438별의 본질인데 7분 인간 드라마는 그 결을 살리지 못함. 회의 결정으로 폐기.

### 단계 2: 단발 7분 풀 콘티 (5/4 v4_short)

**원본**: `06_pd2-5_storyboard.json` (v4_short) + `07_seedance2_prompts.md` (이 폴더에 복사됨)

archived_v2 폐기 후 *결과 중심*으로 재설계된 7분 콘티. Seedance 2.0 영문 prompt 60개 + 한국어 voiceover.

**통제 차원** (전 단계 + 추가):
- ◎ Seedance 2.0 영문 prompt 60개
- ◎ 한국어 voiceover 컷별 명세
- △ 컷당 7초 (Sora 2 8초 한계 내)
- ✗ 컷 안 sub-shot 분해 없음
- ✗ 음절 검수 없음
- ✗ Negative prompt 부분적

**한계**: 60컷 분량의 한국어 립싱크 양순음 누적 부담 + 인터넷강의 영역과 겹침 + 5/5 60초 피벗으로 *최종 운용 영상까지 진입 안 됨*.

### 단계 3: 60초 단발 입구 (5/5-6, YouTube 4편 채택)

**원본**: `07_conference_evolution/shorts_pd1.md` (식자공·의병), `shorts_pd3.md` (점심값), `shorts_pd5.md` (별 자매편), 그리고 별자리 1438 (PD2-5 압축본 — sync 대기)

각 60초 / 6-10컷 / 컷당 4-8초. *Sora 2/Seedance 2.0 단일 클립 한계*에 맞춰 컷 길이 미리 설계.

**통제 차원** (전 단계 + 추가):
- ◎ 컷당 4-8초 (단일 클립 한계 미리 흡수)
- ◎ Negative prompt 컷별 명세 (얼굴 X, 한자 환각 X, 일제 상징 X 등)
- ◎ 후처리 노트 (워터마크·자막·고증 출처)
- ◎ 음성 톤 명세 (50대 남성, 다큐, 차분, 음절당 0.28-0.32초)
- △ 음절 검수 (총 음절 정도, 컷별 음절/초는 단계 4에서 도입)
- △ sub-shot 부분적 (CUT 02 사진+동그라미+탭 같은 multi-beat 일부)

**채택률**: 6편 시도 → 4편 YouTube 게시. 페르소나별 채택률은 `09_outputs/README.md` 참조 — PD1 역사덕후 2편이 가장 강함.

### 단계 4: 시즌 1부 3분 12컷 (5/7, 콘티 정점)

**원본**: `episode_001_hbm_kor.md` (이 폴더에 복사됨)

"한국의 시계" 시즌 1 1부 12컷 풀 콘티. 컷당 15초 + 컷 안 3 sub-shot + 카메라 무빙 + 음절 검수 + Negative prompt + 후처리 노트.

**통제 차원** (전 단계 + 추가):
- ★ 컷 안 sub-shot 3개 분해 (Beat 1 / Beat 2 / Beat 3)
- ★ 컷 안 카메라 무빙 명시 (push-in / dolly / orbit / dissolve 등)
- ★ 음절-시간 검수표 (컷별 음절/초 + 침묵 비율)
- ★ Sound 디자인 컷별 명세 (디제틱 + ambient + tone -dB 단위)
- ★ Negative prompt 컷별 (no readable text / no logos / no faces / no clichés)
- ★ 후처리 환각 차단 명세 (텍스트 100% AE 후처리 / 로고 100% 후처리 / 인물 얼굴 0건)

→ **15초 컷 1개에 들어가는 통제 차원이 7배 깊다**. 60컷 7분보다 12컷 3분이 *사실의 단단함*에서 정점.

## 7분 콘티의 한계

`06_pd2-5_storyboard.json`은 PD2-5 1438년 별의 7분 풀 콘티. 시퀀스 7개·약 60컷으로 분해. 단:

- **AI 영상 prompt가 없음** — 시각 명세만 있고 도구별 prompt는 후속 작업 (`07_seedance2_prompts.md`)
- **컷 7초 단위는 Sora 2 일관성 한계** — Sora 2 8초 일관성 붕괴와 정확히 충돌
- **한국어 립싱크 양순음 위험 분산** — 60컷 분량이면 양순음 구간만 누적 처리해도 후처리 부담 큼
- **음절 검수 없음** — 한국어 5-6 음절/초 기준은 회의 진화 후에 도입된 룰

archived_v2_human_drama 폴더(원본 `runs/cycle-001/archived_v2_human_drama/`)는 *이 7분 콘티의 인간 드라마 중심 버전*. 이천 개인사 중심 서사를 길게 풀어낸 것이 결을 살리지 못했다 — *측정·검증 중심*이 1438별의 본질이라는 회의 결정으로 폐기.

## 60초 콘티의 압축

`07_conference_evolution/shorts_pd1.md`, `shorts_pd3.md`, `shorts_pd4.md`, `shorts_pd5.md`가 60초 6편 시도의 콘티 결과물. 각각 6-10컷 분량.

60초 콘티의 본질:
- **컷당 4-8초** — Sora 2/Seedance 2.0 단일 클립 한계
- **6-10컷** — 60초 안에 *한 발견*만
- **첫 3초 훅 의무** — 모바일 스크롤 차단
- **마지막 5초 "다시 본다" 워드마크** — 채널 정체성 누적

이 형식은 5/6 Conference 002에서 폐기 폭격을 받음 — 데이터 환산만, 추상 마무리, 익숙한 사실 + 새 해석 시늉 등이 60초로 압축돼도 결이 살지 않음.

## 3분 12컷의 부활 — 컷-레벨 디렉터의 정점

`episode_001_hbm_kor.md`는 시즌 1부 "우리는 어디 서 있는가"의 12컷 풀 콘티 (180초, 컷당 15초).

이 콘티의 형식 (이전 단계 누적 룰의 결정체):

```
[Aspect ratio] 16:9, 1920x1080
[Duration] 15 seconds
[Style] Documentary observational, restrained EBS tone, ...

[Scene] Three-beat continuous camera move.
Beat 1 (0-5s): ...
Beat 2 (5-10s): ...
Beat 3 (10-15s): ...

[Composition] Center-weighted throughout. Lower third clean for caption overlay.

[Voiceover - Korean, calm middle-aged male, slow steady cadence] "..."

[Sound] ...

[Negative] No readable text on phone screen, no readable chip labels, no brand logos generated, ...
```

**구성 요소**:
- 시각 명세 + 카메라 무빙 (sub-shot 단위)
- 컴포지션 (캡션 영역까지 명시)
- 한국어 voiceover (음절 검수 5-6/초)
- 사운드 디자인
- **Negative prompt** (환각 차단 핵심)

→ *컷이 곧 사실의 닻*이라는 메타 가설 1의 정확한 형식. 60컷보다 12컷이지만 *각 컷의 깊이*가 7배 깊다.

## 후처리 노트 (AI 환각 차단)

12컷 콘티의 마지막 섹션은 환각 차단 명세:

- 모든 텍스트 (한국어·영문·숫자) — 모델 X. After Effects 후처리 합성
- 회사 로고 (NVIDIA·삼성·SK) — 모델 X. 후처리 합성 또는 추상 형태 유지
- 실제 인물 얼굴 — 0건. 실루엣·뒷모습·손·매크로만
- 칩 매크로 — 가짜 디테일 OK (구체 모델 명시 X)
- 자막 출처 표기 — 영상 끝 0.5초 + 더보기란
- 좌상단 워터마크 전 컷 공통: "재현 영상 + AI 생성"

→ 이 명세가 컷마다 적용된다. 컷 단위 환각 차단이 곧 EBS 신뢰성 — Conference 003 → 004의 픽션 폭로 사건의 직접 학습.

## 이 단계의 의미

이 단계가 메타 가설 1(컷-레벨 디렉터)의 *직접 증거 폴더*다. 컷의 깊이가 다르면:

| 깊이 | 결과 |
|---|---|
| 시각 명세만 | 도구별 재해석 발생 → 결의 일관성 깨짐 |
| Seedance 영문 prompt | 도구 일관성 OK, 한국어 voiceover 동기 약함 |
| sub-shot + 카메라 무빙 + 음절 검수 + Negative + 후처리 노트 | 컷이 사실의 닻 — 한 컷 안에서 호흡·시각·검증 모두 통제 |

→ AI 다큐 PD의 핵심 역량은 시각 디렉팅이 아니라 *컷 단위 모든 차원의 동시 명세*. 현장이 없으니 컷 안에서 호흡·검증·후처리까지 모두 결정해야 한다.

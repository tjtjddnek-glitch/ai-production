## 이 단계의 의미

디벨롭은 *작업 가능한 산출물*이 처음 나오는 단계. 이전 단계는 모두 *결정의 기록*이지만, 디벨롭부터는 영상 제작자(편집·CG·내레이션)에게 넘길 수 있는 시퀀스 분량의 명세가 나온다.

이 단계가 끝나면 다음은:
- 7분 콘티(컷 단위 분해)
- 60초 압축(컷 6-10개)
- 또는 시즌 바이블(8부작 구조 + 1부 콘티)

이 사이클은 7분 콘티 시도 → 60초 피벗 → 3분 12컷 → 시즌형 4부작 12컷으로 진화.

## 결정 narrative

### 7분 콘티의 한계

`08_storyboards/06_pd2-5_storyboard.json` (PD2-5 1438년 별 7분 풀 콘티)이 5/4 작성됨. 그러나:

1. **인터넷강의 영역과 겹침** — CP 평가가 PD1-5에 대해 짚었던 "교과 콘텐츠 보강 성격"이 7분 다큐에서도 발생
2. **AI 제작 한계** — Sora 2 8초 일관성 붕괴 + 한국어 립싱크 양순음 부자연이 7분 단위에서 누적되면 EBS 신뢰성 직격
3. **60초 입구라는 새 가능성** — 같은 결을 6-10컷으로 압축하면 *대표 장면*만 추출 가능. 시청자에게 "더 보고 싶다"를 만들 수 있음

5/5 결정: **60초 6편 동시 업로드** + 채널 신설.

## 5/5 60초 후보 재선정 — `60s_pilot_topics.md`

7개 디벨롭 안과 별개로 *60초 입구로 가장 적합한 10개*를 EBS 교양 CP·숏폼 다큐 PD·AI 영상 프로듀서 관점에서 다시 뽑음.

4. 평균은 왜 우리를 속일까 (PD3-5 압축)
5. 냉장고 문을 열면 왜 생각이 사라질까 (문간 효과)
6. 왜 사람은 자기 목소리를 싫어할까 (뼈전도)
7. 지도는 왜 항상 북쪽이 위일까 (관점 전환)
8. 미세먼지 한 알의 여행 (입자 POV)
9. 우리는 왜 비 오는 날 냄새를 좋아할까 (페트리코어)

**숏리스트 3**: 와이파이 / AI 노년 / 1초 안의 몸

## PITCHES_SUMMARY.md — 7개 한눈에

`PITCHES_SUMMARY.md`는 5/4 7개 디벨롭의 통합 요약. 이게 7분 → 60초 피벗 결정의 *원료 시트*. 다음 단계로 가기 전에 7개의 결을 한 표에 모아둔 것.

이 시트의 마지막 섹션 "인사이트 시그널"은 메타 가설 검증 데이터를 자연 발생적으로 수집:

> 1. 인간(친구) 픽 vs CP(AI) 픽 overlap 40% — 같은 30개를 두고 평가 기준이 다름
> 2. 인간은 '시리즈 일관성·테마 묶기'를 평가 가중치 위에 둠 (PD3 3개 통과, 고령·인구 결로 7개 묶음)
> 3. CP는 'AI 필연성·차별성·STRONG 점수'를 가중치 위에 둠
> 4. PD3·PD4 'AI 안 써도 되는 다큐' 위험 → 디벨롭에서 본질 재설계로 보강
> 5. WEAK→STRONG 자생 보강: PD4-1이 CP WEAK(7.05)에서 페르소나 자체적으로 본질 재설계 → ai_necessity 5→9
> 
> → 메타 가설 2번('워크플로우 설계 = PD 채용') 검증 시그널 첫 캐치.

## REVIEW_SHEET.md — 친구 직접 점수 시트

`REVIEW_SHEET.md`는 30개 로그라인 + 친구 직접 점수 매기는 빈 칸 시트. 5축(공공·고증·학습·확장·AI필연) 가중치를 CP와 동일하게 두고 *친구 자기 점수*를 받기 위한 도구.

이 시트의 의도는 **CP·인간 평가 차이의 정량화**. CP 점수와 친구 점수의 갭이 큰 항목 = 다음 사이클 페르소나 system prompt 조정 시그널.

```
weighted = 0.25*공공 + 0.20*고증 + 0.20*학습 + 0.15*확장 + 0.20*AI필연
```

이 시트는 디벨롭 단계 진입 전 픽 시점에 운용된 도구. 인간 픽 결정의 근거 추적용.

## 이 단계의 의미

이 단계는 *진짜 콘티 작업 전*에 결정 정보를 모은 단계. 7분 → 60초 피벗 결정의 직접 근거가 되는 자료들.

시즌형 진화 관점에서 보면 이 단계의 한계가 자연스럽게 드러난다:
- 60초 후보 10개 중 8개가 *생활 과학 / 일상 호기심* 결 — 시즌 8부작으로 확장 가능한 큰 질문이 부족
- 한국 시청자 시의성이 약한 후보 다수 (와이파이·문간 효과·자기 목소리)
- AI 필연성은 강하나 한국 사회 직격이 약함

이 한계가 5/6-7 회의 진화 단계의 *폐기 기준*에 반영된다 — 시의성·마음 직격·시즌 확장성을 회의록마다 누적.

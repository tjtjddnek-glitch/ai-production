## PD1-5 — 보이지 않는 산: 1908년 가을, 두 군대가 서로를 못 본 17일

**컨셉**: 의병 일지의 행군 좌표와 일본군 토벌 보고서의 부대 이동을 같은 한반도 지형 위에 시간 동기화로 겹쳐, '왜 둘은 같은 골짜기를 17일간 오가면서 서로를 단 한 번도 보지 못했는가'라는 시야·통신·지형의 비대칭을 AI 지형 시뮬레이션으로 재구성하는 단편 다큐.

**타겟**: 30~50대 한국 근대사 관심층 + 한국사 학습자

**시퀀스 6개 (18분)**
1. (90초) 콜드 오픈 — 같은 골짜기, 다른 날짜. 빨간 점(의병)·파란 점(일본군)이 시간차 42시간으로 같은 길을 지남
2. (120초) 사료 두 장 — 같은 마을, 다른 언어. 한국사DB·JACAR 사료를 AI 텍스트 정렬로 매칭
3. (360초) 지형 위에 두 군대 — 17일을 5분으로 압축. 국토지리정보원 DEM 위 행군 궤적
4. (240초) 왜 못 봤는가 — 시야 시뮬레이션. 800m 떨어진 두 능선의 1인칭 안개·활엽수림 시야
5. (180초) 통신·물류 비대칭. 일본군 정보 6시간 vs 의병 31시간
6. (90초) 엔딩 — 보이지 않는 것의 무게. 좌표 1,200개

**AI 파이프라인**
- 사료 처리: Claude Opus 4.7 (1M context로 1,400쪽 양측 사료 동시 OCR·정렬, 한문·일본어)
- 이미지: Flux Pro 1.1 (사료·디테일) + Midjourney v7 (분위기)
- 영상: Kling 2.5 (1인칭 시야), Sora 2 (위성 시점), Veo 3.1 (시간 압축)
- 3D 지도: Blender + 국토지리정보원 DEM
- 음성: ElevenLabs v3 한국어 / 음악: Suno v5 / 편집: DaVinci 19

**차별화**: 인터넷강의는 '왜 의병이 졌나' 텍스트 설명. 이 다큐는 '시야·통신 지연·지형'을 1인칭 시뮬로 답함. AI 1M context + Kling 2.5 1인칭 시야는 2025년 이후 처음 가능.

**CP 약점 보완**
- '교과 보강 성격' → 강의가 닿지 못한 1인칭 시뮬레이션·구조 시각화로 차별

## PD2-5 — 1438년의 별: 이천이 본 한양 밤하늘을 AI로 600년 만에

**컨셉**: 조선 천문학자 이천(李蕆)이 1438년 한양 경복궁에 세운 7종 천문기기로 측정했을 별의 좌표를 NASA JPL Horizons로 역산하고, AI 영상으로 그날 밤하늘을 시청자가 직접 보게 한다.

**타겟**: 30~50대 EBS 다큐 시청자 + 천문 입문자

**시퀀스 7개 (18분)**
1. (90초) 오프닝 — 600년 전의 좌표값 한 줄. 1438년 음력 7월 16일, 위도 37.57°N
2. (180초) 이천과 7종 천문기기 — 결과 중심 소개 (간의·혼천의·자격루·앙부일구·일성정시의·규표·소간의)
3. (150초) 복원 결정 — JPL Horizons 쿼리 UI 등장. 위도·경도·날짜 입력 → 그날 좌표 출력
4. (240초) 1438년 한양 하늘 — Sora 2 + Skyfield로 직녀·견우·백조자리 정확한 적위·적경 배치
5. (180초) 정밀도 검증 — 0.1도. 티코 브라헤(1546~1601)와 비교
6. (150초) 천상열차분야지도 vs JPL Horizons — 평사도법 겹침. 일치=흰색, 어긋남=빨간 점
7. (90초) 엔딩 — 다음 별을 보는 사람들. 보현산 천문대와 분할

**AI 파이프라인**
- 좌표 계산: NASA JPL Horizons API + Skyfield (Python) + Stellarium 3중 검증
- 영상: Sora 2 (밤하늘) / Veo 3.1 (간의대 부감) / Kling 2.5 (인물)
- 3D: Blender 4.2로 7종 기기 디지털 트윈
- 음성: ElevenLabs v3 / 음악: Suno v5 (조선 궁중악 모티프)

**차별화**: 박물관식 유물 외관 X, 위인전 톤 X. '결과 중심 재현'은 한국·해외 어디에도 선례 없음. Kurzgesagt 깔끔 인포그래픽도 BBC 화려한 재현도 아닌 'EBS만의 한국 과학사 결과 중심'.

**CP 약점**: STRONG 8.65로 약점 거의 없음. 별 좌표 환각 위험은 분리 설계로 차단(Sora가 풍경, Skyfield가 별 위치, 후합성).

## PD3-6 — 380조와 0.72: OECD 16개국 출산 정책 회귀 분석

**컨셉**: 한국이 20년간 380조 원을 썼지만 합계출산율은 0.72까지 내려왔다. OECD 16개국 정책 지출·구조·효과를 같은 도표 위에 올려, '돈을 어디에 어떻게 쓴 나라가 효과를 봤는가'를 다중 회귀로 분리해 보는 60분 단독 데이터 다큐.

**타겟**: 30~50대 정책 관심층 + 정책 입안·연구·언론 종사자

**시퀀스 7개 (60분)**
1. (240초) 두 개의 숫자. 380조 곡선과 0.72 곡선이 정반대로 갈라지는 순간
2. (360초) 16개국, 같은 축에 올리다. 4축(가족 지출 %, 육아휴직 소득대체율, 0~2세 공공보육, 출산율)

4. (360초) 한국의 380조 — 어디로 갔나. Sankey 다이어그램 (간접 70%, 보육·돌봄 35%, 출산장려금 15%)
5. (540초) 당신이 쓴다면 — 인터랙티브 정책 시뮬레이터. 5개 슬라이더 + OECD 16개국 매칭
6. (360초) 한 가지 빠뜨린 것 — 신뢰. OECD Trust Index 상관 (인과 X)
7. (240초) 닫는 자막 — 시청자에게 위임

- 회귀: Python statsmodels + Claude (다중 회귀 + 로버스트 체크 + 잔차 분석)

- 영상: Sora 2 + Veo 3.1 + Kling 2.5
- 음성·음악·편집: ElevenLabs v3 + Suno v5 + DaVinci

**차별화**: 기존 KBS·SBS·뉴스타파는 단순 비교. 이 기획은 (a) 다중 회귀로 변수별 효과 분리, (b) Sankey + 산점도로 '돈이 간 곳' vs '효과 큰 곳' 어긋남, (c) 시청자 직접 조작 인터랙티브 — '학술 방법론을 시각·인터랙션으로 번역'.

**CP 약점 보완**

## PD4-1 — 1950년생, 세 도시의 50년: AI 가상 코호트 식탁

**컨셉**: 한국·일본·핀란드의 1950년생 여성 1만 명을 AI 코호트로 시뮬레이션한 뒤, 그 분포에서 가장 통계적으로 평범한 세 명을 가상 출연자로 합성해 75세 아침 식탁을 동시에 열어보는 AI 다큐 실험.

**타겟**: 30~60대 EBS 다큐 시청층 + 정책 담당자

**시퀀스 7개 (28분)**
1. (120초) 프롤로그 — '평균적인 75세 여성'은 존재하지 않는다. 1만 점 분포에서 평균 한 점은 아무도 없는 위치
2. (240초) 코호트 시뮬레이션 — 1만 명의 50년. 마르코프 체인 분기 (한국 사별·자녀 동거, 핀란드 이혼·1인가구, 일본 그 사이)
3. (180초) 의외의 지리 — **헬싱키가 아니라 카이누 숲, 도쿄가 아니라 사가, 서울이 아니라 정선**. 분포의 모드는 변두리
4. (240초) 가상 출연자 합성 — Hedra + Sora 2로 정선 김OO, 사가 タナカ, 카이누 Aino. 25/45/65/75세 시간 일관성 사진
5. (360초) 같은 아침 7시 — 세 식탁 6분 롱테이크. 정선: 밥 반 공기·약 8알·KBS1 / 사가: おかゆ·매실·NHK / 카이누: 호밀빵·청어·라디오 침묵
6. (240초) 분포 다시 보기 — 이 셋은 누구의 평균인가. 정선=자녀 동거 30% 위치, 사가=우울 척도 상위 25%, 카이누=사회 접촉 하위 15%
7. (180초) 에필로그 — 1만 명의 식탁

**AI 파이프라인**
- 코호트: Claude Opus 4.7 + Python pandas/scipy + Claude Code 에이전트 (마르코프 체인)
- 가상 출연자: Hedra Character-3 + Flux (시간 일관성) + Midjourney v7
- 영상: Sora 2 (6분 식탁 롱테이크) + Kling 2.5 (인서트) + Veo 3.1 (분포 시각화)
- 음성: ElevenLabs v3 한·일·핀

**차별화**: KBS '세계의 식탁'·NHK 동시중계와 본질적으로 다른 3가지. (1) 통계 분포에서 추출한 가상 코호트가 출연자, (2) '동시 중계' → '동시 분포 시각화' (1만 명 + 세 점 줌인), (3) **수도가 아닌 통계 모드 위치(정선·사가·카이누)** — PD4 강점 '의외의 지리' 회복.

**CP 약점 보완** (가장 큰 본질 재설계 — WEAK 7.05에서 출발)
- AI 역할 약함 → '코호트 1만 명 시뮬 + 가상 출연자 합성'으로 전면 교체. ai_necessity 5→9
- KBS 단골 포맷 → '동시 분포 시각화' + 인터랙티브 웹 컴패니언
- '의외의 지리' 부재 → 수도 3곳 → 통계 모드 변두리 3곳 교체

## PD5-1 — AI는 한국 노년을 어떻게 상상하나: 8개 모델 vs 진짜 70대

**컨셉**: 글로벌 8개 생성 AI에게 '한국 노인'을 그려달라고 하면 어떤 그림이 나올까. 그 그림을 70대 1인가구 당사자가 직접 보고 반박한다 — AI 편향을 시청자가 눈으로 잡는 메타 다큐 1편.

**타겟**: 30~50대 EBS 다큐층 + 65세+ + 미디어 리터러시 청년

**시퀀스 6개 (25분)**
1. (90초) 프롬프트 입력. 8개 모델 동시 입력 (DALL·E 3, Midjourney v7, Flux Pro, SD 3.5, HyperCLOVA X, Imagen 3, Ideogram 3, Kling Image)
2. (180초) 8분할 출력 그리드. 한복 87% / 시골 72% / 미소 65% / 흑백 41% / 스마트폰 0장
3. (480초) **split-screen 메타 — AI 출력 vs 당사자 반박**. 좌측 AI / 우측 70대 시청자 일상 (운동복·카톡·헬스장)
4. (240초) Latent space 시각화 — t-SNE 2D 임베딩. 학습 데이터 출처 분포 (흑백 사진·1980 다큐 스틸·관광 한복)
5. (240초) **Reverse Generation** — 출연자 사진 40장으로 SD 3.5 LoRA 5분 학습 → 운동복+스마트폰 70대 이미지 출력
6. (270초) 엔딩 — 시청자 질문. 출연자 본인이 셀카 인스타 업로드

**AI 파이프라인**
- 이미지 생성 8개 모델: 동일 프롬프트·시드·해상도, 32장
- 자동 라벨: CLIP + BLIP-2 → 12개 카테고리
- 임베딩: OpenCLIP → UMAP/t-SNE
- 영상: Sora 2 + Kling 2.5 (latent space 추상)
- LoRA: SD 3.5 + RunPod RTX 4090 (1시간)
- 진위 분리: 인터뷰는 진짜 카메라(iPhone 15 Pro/Sony FX3), AI 생성물은 워터마크

**편향 측정 (가벼운 3종)**
- 키워드 빈도 (CLIP 자동 라벨, 12 카테고리)
- Latent space 클러스터링 (Silhouette score)
- 통계청 데이터 vs AI 출력 갭 (도시 73%, 스마트폰 81%, 한복 일상 1% 미만)

**차별화**: (1) 'AI를 쓴 다큐' 아닌 'AI를 분석한 다큐' — 메타 본질, (2) 당사자 split-screen 반박, (3) LoRA 학습 시연 — 미디어 리터러시 새 포맷, (4) HyperCLOVA X 1개 포함 — 'Made in Korea AI'도 같은 편향인가, (5) 차분한 톤 (자극 X) — '우리가 인터넷에 올린 모습이 거울'

**CP 약점 보완** (고증 7점, 가벼운 버전)
- 학술 메트릭 5종 → 직관 3종으로 압축
- AI 윤리 자문 30분 영상 포함 — 자문 학자가 '직관 시각화이지 검증 아니다' 직접 말함
- 통계청 객관 앵커 + 시드 고정 재현 가능성

# 02. 로그라인 단계 — 5 PD × 6 = 30개

> 5명의 PD 페르소나가 같은 리서치 입력을 받고 각자 6개씩 로그라인을 생성.
> 이 단계가 메타 가설 2(워크플로우=PD 채용)의 *원료*다.

---

## 단계 위치

```
[1] 리서치
[2] 로그라인 (5 PD × 6 = 30) ★  ← 이 단계
[3] CP 평가
[4] 인간 픽
```

## 이 단계의 의미

5명 PD가 같은 입력을 받아도 *결이 다른 6개씩*을 뽑는다는 것이 메타 가설 2의 전제 데이터. PD1은 미시사·사라지는 직업, PD3은 통계 리터러시·정책, PD5는 메타 다큐 — 페르소나마다 *질문하는 방식* 자체가 다르다.

다음 사이클 실험: 같은 페르소나 system prompt에 모델만 다르게(Sonnet 4.6 vs Opus 4.7 vs DeepSeek V3) 돌렸을 때 *동일 결의 강도가 어떻게 달라지는가*. 이게 "페르소나 = system prompt" vs "페르소나 = system prompt + 모델"의 분리 실험.

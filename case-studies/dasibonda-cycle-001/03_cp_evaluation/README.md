# 03. CP 평가 단계 — 5축 가중 평가

> 30개 로그라인을 CP 시니어 페르소나가 5축 가중으로 평가.
> 단순 점수가 아니라 *왜 OK / WEAK / KILL인가*의 코멘트가 본질.

---

## 단계 위치

```
[1] 리서치
[2] 로그라인
[3] CP 평가 ★  ← 이 단계
[4] 인간 픽
```

---

## 평가 5축 + 가중

```
weighted_score = (
    publicness    * 0.25  +   # 공공성
    veracity      * 0.20  +   # 고증
    learning_intent * 0.20  + # 학습 의도성
    scalability   * 0.15  +   # 확장성
    ai_necessity  * 0.20      # AI 필연성
)
```

CP 시니어(20년차) 페르소나의 가중. 공공성·고증·학습 의도가 무거움 — 자극·재미·뷰가 가벼움. **EBS 본채널 결**의 명시화.

---

## 결과 분포

- **STRONG** (8.20+): PD1-1, PD1-3, PD1-6, PD2-1, PD2-2, PD2-5, PD3-2, PD3-3, PD3-4, PD3-5, PD4-3, PD5-1, PD5-3, PD5-4 — 14개
- **OK** (7.0-8.19): 12개
- **WEAK** (6.0-6.99): 3개 (PD4-1, PD4-4, PD5-5)
- **KILL** (-5.99): 1개 (PD4-5)

→ **CP의 top 5 추천**: PD1-6, PD2-5, PD3-4, PD4-3, PD5-1

## CP 평가의 톤 — 폐기 코멘트 인용

CP 코멘트는 단순 점수가 아니라 *EBS 결과 어긋나는 지점*을 짚는다. 폐기 사례 인용:

**PD4-5 (튀니지·강원도 손빵의 인류학) — KILL 5.95**
> "주제 자체가 정체성에 약함. 손빵의 비교는 학술적으로 흥미로울 수 있지만 EBS 한국 시청자에게 *왜 봐야 하는가*에 대한 답이 약함. 게다가 인류학 비교 다큐는 KBS·MBC·EBS 자체에서 이미 충분히 했고, 본 페르소나의 비교 우위가 살지 않는 영역."

**PD5-5 (1만 명 동시접속 1인가구 토론) — WEAK 6.55**
> "1만 명 동시접속이라는 규모는 흥미롭지만 EBS 다큐 포맷이 아니라 *이벤트* 포맷. 다큐로 만들면 라이브의 본질이 죽고, 라이브로 가면 EBS 편성·아카이브 자산화 모두 어긋남. 페르소나 강점인 메타 다큐 결도 안 살음."

**PD4-1 (서울·도쿄·헬싱키 75세 여성 식탁) — WEAK 7.05**
> "KBS '세계의 식탁' / NHK 동시중계 단골 포맷에 매우 가까움 — '글로벌 동시 시계' 발상의 차별화가 약함. AI 역할도 모핑·자막 정도라 '필연성'이 가장 약한 안건 중 하나. (...) 한국·일본·핀란드 차이는 흥미롭지만 다큐 1편으로 압축 시 인물 1인 한정 위험."

→ PD4-1은 인간이 CP WEAK를 *무시하고* 픽함 (`04_human_cut.json`의 일탈 기록 참조). 디벨롭 단계에서 PD 자체적으로 본질 재설계 → STRONG급 콘텐츠로 전환됨 — 이게 페르소나 자체 보강 능력의 자연 캐치.

## STRONG 코멘트 인용 — PD2-5 1438년의 별

> "조선 천문학(간의·혼천의) '결과 중심' 재현이라는 각도가 매우 강함. 한국 과학사 콘텐츠는 늘 유물 외관 소개에 머물렀는데 이건 '그가 본 별'을 NASA JPL Horizons로 역산해 같이 본다는 본질 시각화. AI 천체역학 시뮬레이션이 '필연성' 충족. 한영호·한국천문연구원 자문 견고. EBS 비교우위 영역. 시즌화·교과 연계도 강함. PD2 안에서 가장 강함."

이 평가가 이후 모든 사이클의 *기준점*이 된다. 이후 회의록에서 반복적으로 인용되는 "1438년 별 결" — *작은 사물 → 깊은 검증 → 반전 발견*은 사실 이 CP 평가에서 처음 정립된 결.

## 이 단계의 의미

CP 평가는 **인간이 픽하기 전 1차 필터**다. 인간이 30개를 다 읽지 않아도 STRONG/OK/WEAK가 표시된 상태에서 결정 가능 — 시간 압축. 단, 이 필터가 *너무 강하면* 인간의 직관·시리즈 의도가 자유도를 잃음.

`04_human_cut.json`의 결과는 인간이 CP top5와 40%만 일치 — 즉 CP 평가는 *참고는 하되 종속하지 않는다*는 것이 인간의 운영 시그널. 메타 가설 2의 핵심 데이터.
